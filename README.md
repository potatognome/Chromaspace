# Colour System: 36×5×5 Taxonomy

A comprehensive, deterministic colour taxonomy for design, data, and computation. This project implements a 36-hue, 5-saturation, 5-luminance system with semantic naming, JSON export, and XKCD nearest-colour matching.

- **900 unique colours**: 36 hues × 5 saturation × 5 luminance
- **Semantic names**: e.g., `red-warm vivid-bright`
- **Machine fields**: hue, variant, sat, lum, HSV, RGB, XKCD match
- **JSON output**: All 900 entries, ready for use

See `docs/overview.md` for details.
